# w3m

## basic navigation
* `c` show current address
* `u` show target address
* `esc+l` popup with all links
* `shift-b` back
* `shift-u` url prompt

## tabs
* `{}` move between tabs
* `shift-t` new tab
* `ctrl-t` open link in new tab
* `ecs-t` to tab screen
* `esc-t` `shift-D` to close tab

## bookmarks
* `esc-b` view bookmarks
* `esc-a` add to bookmarks

## 
* `shift+m` open tab in ext browser
